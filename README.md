# gh pr-mark-as-viewed

A GitHub CLI extension to mark PR files as viewed.

## Installation

```bash
gh extension install happylinks/gh-pr-mark-as-viewed
```

## Usage

```bash
# Mark all files in a folder as viewed
gh pr-mark-as-viewed src/components/

# Mark a single file as viewed
gh pr-mark-as-viewed README.md
```

Run from a branch that has an open PR.
